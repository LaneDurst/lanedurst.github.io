    • LaneDurst.Github.io is my personal portfolio website, designed and implemented, entirely by me.

    The website was implemented using Github's built-in Pages function, which allows each user to create a personal static-website hosted by github. [Documentation Here]

    In an effort to hone my CSS and HTML skills, I elected to forgo the recommended Jekyll libraries and frameworks such as react. Instead, this website is coded by hand in pure HTML/CSS.

    As a result, I have uncovered various features of CSS that I was previously unaware of. For instance, this website will change styling on a small enough screen! Try loading it on your phone, you will notice the homepage has 2 columns instead of 4. This was accomplished using the viewport meta tag on the HTML file and an @media query in the CSS file.

    I also used portfolio.adobe.com to find inspiration for many aspects of the website. This is a free serivce offered by adobe that showcases portfolio websites from various artists on the web.
