    • RobotStudio AI Assistant

    The RobotStudio software is an offiline robot programming enviornment created by ABB. It serves to allow robot programmers to test changes to various ABB robots without needing access to a physical ABB Robot.

    This project was carried out as part of the LSU Robotics Lab, now known as the riseatlsu lab. It served to conduct research into the effectiveness of AI in assisting non-techinical audiences use the software effectively.

    My key responsibilities on this project were coordination between team members, implementation of the interface, and connectivity to the backend. The interface was created using C# and the RobotStudio SDKs, while the backend (handled by the other team members) was created using a ChatGPT 4 Model with specialized instructions.
